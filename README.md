# BULK MODE — مدرّب التضخيم 💪

تطبيق ويب عربي متكامل (PWA) لمتابعة تمارين الجيم اليومية، مبني على بنية برنامج "BULK MODE" الأصلي مع الحفاظ الكامل على الهوية البصرية الفاخرة (ذهبي على أسود عميق).

---

## 🎯 الميزات المنجزة

### 📅 الصفحة الرئيسية
- بطاقة "تمرين اليوم" التي تكتشف يوم الأسبوع تلقائياً
- مؤشر "جلسة جارية" إذا كان هناك تمرين لم ينته
- قائمة الأيام الـ 7 (دفع A/B، سحب A/B، أرجل A/B، يوم راحة)
- آخر 4 جلسات مع روابط تفصيل سريعة
- شريط إحصائيات سريعة (عدد الجلسات / الأسبوع / المدة الكلية)

### 🏋️ صفحة اليوم (تفاصيل التمرين)
- عرض كامل للتمارين مع أهداف السيت/التكرار/الراحة
- رقم "أفضل وزن" لكل تمرين
- زر مباشر لبدء الجلسة
- محتوى يوم الراحة كاملاً (المشي، التمدد، النوم...)

### 💪 جلسة التمرين الحية
- مؤقت إجمالي للجلسة (يعمل في الخلفية)
- شريط تقدم حسب السيتات المنجزة
- إدخال **الوزن (كجم) + التكرارات** لكل سيت بحقول كبيرة مناسبة للجوال
- زر ✓ لتأكيد إنجاز السيت
- **استدعاء أوزان آخر جلسة تلقائياً** كقيم افتراضية لتسريع الإدخال
- مؤقت راحة منبثق + **اهتزاز + صوت تنبيه** عند الانتهاء
- أزرار سريعة لتغيير مدة الراحة (30/60/90/120/180) و +/- 15ث
- إضافة سيت إضافي بضغطة زر
- خانة ملاحظات حرة عن الجلسة
- حماية من إغلاق الصفحة بالخطأ
- Wake Lock — الشاشة لا تنطفئ أثناء التمرين

### 📊 السجل والإحصائيات
- سجل كامل مجمَّع بالشهر
- تفاصيل أي جلسة قديمة (السيتات بالأوزان والتكرارات)
- **مقارنة تلقائية بين كل سيت والجلسة السابقة** (📈 +2.5كجم / 📉 -1تكرار / =)
- مقارنة الحجم الكلي بآخر مرة (مجموع الوزن × التكرار)
- إحصائيات: عدد الجلسات، السيت الكلي، الوزن المرفوع، الساعات، السلسلة المتصلة
- توزيع التمارين (دفع/سحب/أرجل) بصرياً
- 🏆 أفضل أرقام شخصية للتمارين الرئيسية (Chest Press، Lat Machine، Leg Press)
- زر **تكرار اليوم** (يبدأ جلسة جديدة بنفس البرنامج)

### 💾 إدارة البيانات
- حفظ كامل في `localStorage` (يبقى بعد إغلاق المتصفح)
- **تصدير** نسخة احتياطية JSON
- **استيراد** نسخة احتياطية
- حذف كل البيانات (مع تأكيد)
- يعمل **بدون تسجيل دخول** بالكامل

### 📱 PWA — تطبيق على الجوال
- `manifest.json` كامل ⇒ يمكن تثبيته من المتصفح كتطبيق مستقل
- `sw.js` (Service Worker) ⇒ يعمل **بدون إنترنت** بعد التحميل الأول
- وضع `standalone` — بدون شريط المتصفح
- معالجة `safe-area` لجوالات iPhone (notch)
- اهتزاز Vibration API
- أيقونة SVG داخلية (لا حاجة لملفات صور)

### 🎨 الهوية البصرية
- الذهبي (#D4A853) على الأسود العميق (#050608)
- خطوط: Rubik للنصوص، Outfit للعناوين، JetBrains Mono للأرقام
- تأثير حبيبات (grain) خفيف، توهجات (shimmer) ذهبية
- RTL كامل + جوال أولاً
- أزرار كبيرة وواضحة لاستخدامها داخل الجيم

## 🧭 مسارات التنقل (Routes)

التطبيق يستخدم راوتر داخلي بسيط — كل المشاهد في `index.html`:

| المعرّف | الوصف |
|---|---|
| `home` | الصفحة الرئيسية + بطاقة اليوم + قائمة الأيام |
| `day` | تفاصيل يوم تمرين (params: `id` = `push-a` / `pull-a` / `legs-a` / `rest-1` / `push-b` / `pull-b` / `legs-b`) |
| `session` | الجلسة الحية (تستخدم `Store.getActiveSession()`) |
| `session-detail` | عرض جلسة قديمة (params: `id` = معرّف الجلسة) |
| `history` | السجل + الإحصائيات (تبويبتان) |

استدعاء برمجي: `Router.go('day', { id: 'push-a' })`

## 🗄 نموذج البيانات (localStorage)

المفتاح: `bulkmode_v1`

```js
{
  sessions: [
    {
      id: 'id_xxxx',
      dayId: 'push-a',
      dayTitle: 'يوم الدفع A',
      dayType: 'push' | 'pull' | 'legs' | 'rest',
      startedAt: 1715000000000,    // Date.now()
      finishedAt: 1715003600000,
      durationSec: 3600,
      notes: 'نص حر',
      exercises: [
        {
          exerciseId: 'pa1',
          name: 'Chest Press',
          targetSets: 4, targetReps: '8-10', targetRest: 90,
          sets: [
            { weight: 25, reps: 10, done: true },
            { weight: 27.5, reps: 8, done: true },
            ...
          ]
        }
      ]
    }
  ],
  activeSession: { ... } | null,    // جلسة لم تنته بعد
  settings: { defaultRest: 90, soundOn: true, vibrationOn: true }
}
```

### واجهة التخزين `Store`
- `Store.getSessions()` — كل الجلسات مرتبة من الأحدث للأقدم
- `Store.getSessionsByDay(dayId)` — جلسات يوم معيّن
- `Store.getBestForExercise(exerciseId)` — أفضل وزن مرفوع
- `Store.getLastSetsForExercise(exerciseId, beforeTimestamp?)` — آخر جلسة (للمقارنة وتعبئة افتراضية)
- `Store.totalVolume(session)` — مجموع (وزن × تكرار) للجلسة
- `Store.getStats()` — إحصائيات شاملة
- `Store.exportJSON() / importJSON(text) / resetAll()`

## 🧪 كيف تستخدم التطبيق

1. افتح `index.html` على الجوال (أو ضمن TheNewly Publish).
2. على iPhone: افتح في Safari → مشاركة → "إضافة إلى الشاشة الرئيسية".
   على Android: Chrome سيقترح "تثبيت التطبيق" تلقائياً.
3. اضغط **ابدأ التمرين** على بطاقة اليوم.
4. لكل سيت: اكتب الوزن + التكرار، ثم اضغط ✓ — سيبدأ مؤقت الراحة تلقائياً.
5. عند الانتهاء، اضغط **إنهاء الجلسة** — سترى ملخصاً + مقارنة بآخر مرة.
6. ادخل لـ "السجل" ⇒ "الإحصائيات" لمتابعة تطورك.
